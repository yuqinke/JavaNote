JVM
1.JVM内存：程序计数器、方法区、Java栈、本地方法栈、堆
程序计数器：每个线程都需要有自己独立的（线程私有）
方法区：线程共享，储存每个类信息，方法区中有一个非常重要的部分就是运行时常量池，类和接口被加载到JVM后，对应的运行时常量池就被创建出来用来存储编译期间生成的符号引用。
Java栈:储存栈针，每个栈帧对应一个被调用的方法，线程当前执行的方法所对应的栈帧必定位于Java栈的顶部
本地方法栈：本地方法栈则是为执行本地方法（Native Method）服务
堆：被所以线程共享，储存对象以及数组
